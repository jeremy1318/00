La importancia de los numeros de las etiquetas son la clasificacion que tiene cada uno de ellos, para diferenciar de uno al otro. Mientra cada uno tiene su utilidad, se va reduciendo el tamaño con cada uno de ellos

8- Investiga qué hacen y cómo se utilizan las siguientes etiquetas HTML: strong i u sub sup em a span

strong: marca las partes mas importantes de un texto, haciendo enfasis.
i: muestra el texto marcado con un estilo en cursiva o italika.
u:Muestra el texto subrayado.
sub: con este se utiliza de una forma topografica, mas baja, y generalmente mas pequeño, que el tramo principal del texto, un subindice.
sup: en este se muestra mas alto y generalmente mas pequeño que el tramo principal del texto, superindice.
em: es la parte que marca con enfasis las partes mas importantes de un texto, este puede ser anidado
a: crea un enlace a otra pagina de internet, arcgivo o ubicacion dentro de la misma pagina, direccion de correo, o cualquier otra URL
span: Sirve para aplicar estilo al texto o agrupar elementos en línea.

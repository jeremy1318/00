1.2- ¿Para qué sirve la etiqueta ol? Dar un ejemplo en donde lo utilizarías.

la etiqueta ol permite definir listas o viñetas ordenadas ("Ordered List"), bien con numeración o alfabéticamente.
Por ejemplo:

  1. primero
  2. segundo
  3. tercero

1.4- ¿Para qué sirve la etiqueta li? Dar un ejemplo en donde lo utilizarías.

2.1- ¿qué es una url absoluta?

una url absoluta nos proporcionan la URL completa del documento, incluido el protocolo (http: // o https: //), el nombre de dominio y la ruta como necesario. Debe usar una URL absoluta cuando apunta a un documento en la web

2.2- ¿qué es una url relativa?

una url relativa es la ruta a un archivo relativo al actual documento. Las URL relativas se pueden usar cuando se vincula a otro documento en su propio sitio (es decir, en el mismo servidor).
